<strong>Create a new repository on the command line</strong><br>
<br>
touch README.md<br>
git init<br>
git add README.md<br>
git commit -m "first commit"<br>
git remote add origin https://github.com/memboc/apserver.git<br>
git push -u origin master<br>
<br>
